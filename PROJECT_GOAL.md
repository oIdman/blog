# blog — 产品目标与开发状态

本文件用于在任意新开发环境中恢复完整上下文。开始开发前请先阅读 AGENTS.md 和本文件，
不要把目标缩减为当前已经完成的功能。

## 当前技术与运行方式

- 仓库：oIdman/blog
- 运行时：纯前端 HTML/CSS/JS，零外部依赖，支持 Markdown 渲染和简易代码高亮
- 启动：直接用浏览器打开 `src/index.html`
- 架构：单页应用 (SPA)，hash 路由切换页面；可托管 GitHub Pages
- 文章存储：localStorage + 内置示例文章（posts.js）
- 设计语言：浅色极简 Geek 风（程序员博客），低饱和代码色
- 字体栈：Inter / JetBrains Mono（程序员友好）
- 版本管理：`src/js/version.js` 为版本单一事实来源

## 版本详细记录

### v0.1.0 — 项目初始化
- 搭建开发工作流（AGENTS.md、PROJECT_GOAL.md、README.md）
- 基础博客页面：文章列表、Markdown 渲染、响应式布局
- 内置 Markdown 解析器
- 示例文章 "Hello, World"
- 自动化脚本（版本管理、通知、Issue 处理）

### v0.2.0 — 文章编辑器
- Markdown 写作界面（左写右预览的双栏编辑器）
- 编辑器路由：`#new` 新建文章、`Edit` 按钮修改已有文章
- 文章数据持久化到 localStorage，刷新不丢失
- 支持发布、编辑、删除操作
- 用户文章与默认示例文章合并显示

### v0.3.0 — Geek 风完整博客重建
- 整体重构为浅色极简 Geek 风格，采用 hash 路由的 SPA 架构
- 首页：sticky 导航 + banner（CSS 浮动代码字符装饰）+ 全局搜索 + 70/30 双栏布局 + 分页
- 文章卡片 + 分类标签 + 时间 + 阅读量 + 评论数；点击直达详情
- 侧边栏：分类导航（带计数）+ 热门 TOP10
- 文章详情：内置 Markdown 解析 + 代码块复制按钮 + 简易语法高亮（JS/Python/Go/Bash）
- 评论区：localStorage 持久化，每篇文章独立评论列表
- 关于我页面：个人卡片 + 技术栈 chips + 开源项目卡 + 联系方式
- 响应式：<920px 自动单栏 + 移动端汉堡菜单
- 完整设计系统：CSS 变量、Inter/JetBrains Mono 字体栈

### v0.4.0 — 页面内容增强
- 首页：新增统计栏（累计文章/分类/阅读量/评论数），让访问者快速了解博客概况
- 全部文章：独立页面 `#/articles`，不再与首页混淆；全宽布局展示所有文章，保留搜索功能
- 导航修复：点击"全部文章"正确高亮对应导航项，不再指向"首页"
- 关于我：新增技术历程时间线（从大学到至今）；技术栈扩展到 15 个；新增博客数据统计
- 新增 CSS：`.stats-bar` 统计栏组件 + `.timeline` 时间线组件，支持响应式适配
- 版本号升至 v0.4.0

## 长期目标

构建一个轻量、零依赖的个人博客，用 Markdown 写文章，打开浏览器就能看。
核心差异化：零构建、零依赖、即开即用、响应式、Markdown 原生渲染。
