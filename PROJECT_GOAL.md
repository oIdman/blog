# blog — 产品目标与开发状态

本文件用于在任意新开发环境中恢复完整上下文。开始开发前请先阅读 AGENTS.md 和本文件，
不要把目标缩减为当前已经完成的功能。

## 当前技术与运行方式

- 仓库：oIdman/blog
- 运行时：纯前端，零第三方依赖，内置 Markdown 解析器
- 启动：直接用浏览器打开 `src/index.html`
- 文章存储：`posts/` 目录下的 .md 文件，通过 posts.js 数据文件加载
- UI：原生 HTML、CSS、JavaScript；响应式布局
- 架构：静态页面，可托管 GitHub Pages
- 版本管理：`src/js/version.js` 为版本单一事实来源

## 长期目标

核心差异化：零构建、零依赖、即开即用、响应式、Markdown 原生渲染。
